# Fashion Store - Java Full Stack Web Application

## Overview

Fashion Store is a Java Full Stack web application developed during my internship at **Tap Academy (Tap EdTech Pvt. Ltd.)**. The application provides an online shopping platform where users can browse fashion products, view product details, manage their shopping cart, and place orders through a user-friendly interface.

The project demonstrates the implementation of frontend development, backend business logic, database connectivity, and web application deployment using Java Full Stack technologies.

---

### User Features

## System Architecture

The application follows a three-tier architecture:

### 1. Presentation Layer

* HTML
* CSS
* JavaScript
* JSP

Responsible for displaying webpages and interacting with users.

### 2. Business Logic Layer

* Java
* Servlets

Handles authentication, cart operations, product management, and order processing.

### 3. Data Layer

* MySQL
* JDBC

Stores and retrieves application data from the database.

---

## Project Workflow

1. User visits the Fashion Store website.
2. User registers or logs into the application.
3. User browses available products.
4. User views detailed product information.
5. User adds products to the shopping cart.
6. User updates cart items or removes products.
7. User proceeds to checkout.
8. Order details are stored in the MySQL database.
9. User can view previous orders in the Order History section.

---
